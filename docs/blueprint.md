# **App Name**: VerticalSync

## Core Features:

- Employee Profile Creation: Create employee profiles with personal and contract details.
- Basic Payroll Calculation: Calculate net pay based on salary, allowances, and deductions.
- Payslip Generation: Generate PDF payslips for employees.
- Bank Export: Generate CSV for bank transfers.
- User Authentication: Implement basic user authentication.
- Compliance Advisor Tool: AI-driven tool providing recommendations for compliance and legal mandates, to advise the user on regional regulatory guidelines

## Style Guidelines:

- Primary color: orange (#FFA500) to evoke trust and reliability.
- Background color: cream white (#FFFAF0), a desaturated version of the primary hue.
- Accent color: yellow (#FFFF00) for highlighting important actions and information.
- Body and headline font: 'Inter' sans-serif for a modern, neutral, and machined feel, good for both headings and body text.
- Use clean, minimalist icons for key actions and navigation.
- A clean and structured layout for easy navigation and data entry.
